# Tech Stack

## 1. Core Technologies

- **Programming Language:** PHP
- **Backend Framework:** Laravel (version 12.x)
- **Frontend Frameworks/Libraries:**
    - Livewire
    - Tailwind CSS
    - Alpine.js
    - Axios (for client-side HTTP requests)
- **Database:** MySQL
- **Build Tools:**
    - Vite (for frontend asset compilation)
    - Composer (for PHP dependency management)
    - npm (for Node.js package management)
- **Desktop Application:** NativePHP

## 2. Development Tools & Practices

- **Version Control:** Git
- **Testing Framework:** PHPUnit
- **Code Quality:** PHPStan, Laravel Pint (linting/formatting)
- **CI/CD:** Automated "Robot Guard" for linting and unit tests (as described in the Engineering Manifesto).
